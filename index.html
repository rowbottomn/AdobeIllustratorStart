Adobe Illustrator

Adobe Illustrator (AI) is a program for making professional graphics for webpages, print, t-shirts and icons. We are going to take some time to ensure that you are able to create interesting graphics to spice up your webpages rather than just google image searching.

Learning tasks for this unit:

    Hover on images to see a larger size image.

  • Copy making a design from a scanned in piece of work
  • Copy making t-shirt design from the design
  • Copy making poster design from the design

Assessment tasks for this unit:

  • Make a custom bear paw logo from scratch
  • Make a web image set for your bear paw image logo
  • Make a custom image from a bitmap image
  • Make the custom image into a sticker and poster design
